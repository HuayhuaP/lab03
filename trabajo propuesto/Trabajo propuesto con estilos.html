Mi presentación personal

Hola, me llamo Roberto Carlos y soy un desarrollador web apasionado por la tecnología y el aprendizaje. Me gusta crear páginas web dinámicas, interactivas y responsivas que ofrezcan una buena experiencia de usuario.

Sobre mí

Algunas de las cosas que me gustan son: desarrollo de ideas en modelo 3D, jugar videojuegos, escuchar música. También me interesa el arte, la cultura y la arquitectura que cuenta la historia de diferentes países.

Tecnologías y lenguajes de programación

Estas son algunas de las tecnologías y lenguajes de programación que domino o que estoy aprendiendo:

    • HTML: El lenguaje de marcado utilizado para crear la estructura de las páginas web.
    • CSS: El lenguaje de hojas de estilo utilizado para dar estilo y diseño a las páginas web.
    • JavaScript: Un lenguaje de programación utilizado para agregar interactividad y dinamismo a las páginas web.
    • Bootstrap: Un marco de diseño de código abierto que facilita la creación de sitios web responsivos y atractivos.
    • Oracle: Una base de datos relacional utilizada para gestionar y almacenar datos en aplicaciones empresariales.
    • PHP: Un lenguaje de programación del lado del servidor ampliamente utilizado para desarrollar aplicaciones web dinámicas.
    • MySQL: Un sistema de gestión de bases de datos relacional de código abierto ampliamente utilizado.
    • Laravel: Un marco de desarrollo de PHP que simplifica la creación de aplicaciones web complejas.
    • React: Una biblioteca de JavaScript utilizada para crear interfaces de usuario interactivas y dinámicas.
    • MongoDB: Una base de datos NoSQL que almacena datos en formato JSON y es escalable y flexible.

Formación profesional

Tengo un título de técnico en desarrollo web y aplicaciones multiplataforma. He realizado varios cursos online y presenciales para ampliar mis conocimientos y habilidades. He trabajado como desarrollador web freelance y en varias empresas del sector.

Idiomas

Hablo español como lengua materna, inglés a nivel básico. Me gusta aprender idiomas porque me permiten comunicarme con personas de diferentes partes del mundo y acceder a más información y recursos.
